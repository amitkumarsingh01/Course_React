# Basic Course React WebApp

## How to RUN
### Clone the repository
```
git clone "repo"
cd react-course-app
```
### To run the backend
```
cd backend 
npm i
node server.js
```

### To run the frontend
```
cd ..  // Go back to react-course-app
npm i
npm start
```
